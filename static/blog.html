C++ Study

Design is a plan for solving a problem, but leads to multiple solutions. Coupling is degree of interdependence among programming modules. Cohesion is degree of element association within a module. Our goal: High cohesion, low coupling

C++ is composed of 3 languages:

1. Before compilation, preprocessor language modifies the program
2. During compilation, template language adds new type and routines
3. During compilation, - C specifies basic declaration and control flow
- C++ specifies advanced declaration and control flow

Beware of overflow:

General assignment operators only evaluate left-hand side once, ex: -=, +=, *=...
Bit-shift operators, shift bits in integral variables left and right. (left shift is multiply by 2)

Exception throw and catch:

Dynamic multi-level exit allows complex forms of transfers among routines, called exception handling.
Exception handling is more tan error handling.
An exception event is an event that is known to exist but which is ancillary to an algorithm.
A routine can explicitly specify if it does or does not have alternate outcomes.

Enumeration:

An enumeration is a type defining a set of named literals with only assignment, comparison, and conversion to integer, ex: enum Days {Mon, Tue, Wed, Thu, Fri, Sat, Sun}
First numerator is implicitly numbered 0; thereafter, each is implicitly numbered +1 the previous enumerator.
Enumerators can be explicitly numbered: enum{ A = 3, B, C = A - 5, D}
Enumerator must be unique in block.

Pointer & Reference

pointer/reference is a memory address.
pointer to a string literal must be const.
Difference between pointer and reference is an extra implicit deference
Reference is like a constant plain pointer, no reference assignment after declaration, must be initialised when declare, have no address so there is no reference to reference or pointer to reference. If we get the address of a reference, it is actually the address of the object that reference point to. Also, array of references is NOT allowed.
Pointer/reference type constructor is NOT distributed across the identifier list.
Do not use pointer arithmetic to subscript arrays.
